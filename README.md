# khan_ayasha_002697806_WebDesign_assignments

Web design coursework and exercises organized by assignment folder. Each assignment is self-contained and can be opened directly (static HTML/CSS/JS) or run as a Node/React app where noted.

## Folder guide
- `Assignment 1` - Static HTML/CSS site (home decor shop).
- `Assignment 2` - Static portfolio site with form, media, and sections.
- `Assignment-3` - HTML/CSS/JS exercises (table layout + scripting) and related assets.
- `Assignment 4` - HTML form with supporting JS.
- `Assigment5` - Multi-page site using SCSS (variables, mixins, nesting, etc.).
- `assignment 7` - JS mini apps (calculator, stopwatch).
- `Assignment_8/Assignment 8` - Node/Express/Mongoose app.
- `Assignment9` - MERN-style split app: backend API + React frontend.
- `Assignment 10` - React app (Create React App).

## Run / view assignments
- Static assignments (1, 2, 3, 4, 5, 7): open the main `.html` file in a browser.
- Assignment 8 (Node/Express/Mongoose):
  - `cd "Assignment_8/Assignment 8"`
  - `npm install`
  - `node app.js`
- Assignment 9:
  - Backend: `cd Assignment9/backend`, `npm install`, `npm start`
  - Frontend: `cd Assignment9/frontend/my-app`, `npm install`, `npm start`
- Assignment 10 (React):
  - `cd "Assignment 10"`
  - `npm install`
  - `npm start`

## Notes
- Assignments 8 and 9 require a local MongoDB instance if you want to exercise the database features.
- Each assignment folder may include its own README with more details.
